About Bookie - version Unknown

Are you wondering why it's saying Unknown, it's because we're on a rolling version at the moment!
This version is actually a re-write of the old one that used:

  • Server generated pages
  • Colander for Schemas + Deform + Deform Bootstrap for Form generation


New version features

  • Fully driven by API & javascript by AngularJS using the backend only for JSON
  • Bourne for serialization of objects
  • WTForms with schema generation from database models for data validation in API
  • Mozilla Cornice library for API services


All versions use:

  • Mako for Templating server generated pages
  • Pyramid framework for backend
  • SQLAlchemy for DB / ORM
  • Alembic to provide DB Migrations
  • CSS / Script minification via webassets / pyramid_webassets
